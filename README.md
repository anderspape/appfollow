# Project Workspace

This repository now contains two separate projects:

- `focustimer/`: Swift macOS FocusTimer app
- `mcp/`: AppFollow MCP server (Python)

## Quick Start

### FocusTimer

```bash
cd focustimer
swift build
```

### MCP

```bash
cd mcp
APPFOLLOW_API_TOKEN="YOUR_APPFOLLOW_TOKEN" ./run.sh
```
